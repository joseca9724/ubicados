Recomendaciones

¿Tentado por el teléfono?

Evita tenerlo cerca para que no sea objeto de distracciones, como para revisar redes sociales

¡Organiza tu espacio!

Busca tener un espacio justo con lo necesario que ocupes para llevar la clase o estudiar. Así reduces las posibles distracciones.

Elabora una rutina

Es bueno definir una hora para levantarse, bañarse e incluso para sentarse a estudiar.

Busca tu sitio ideal

Es importante adecuar un espacio para estudiar. Debe tener buena ventilación, iluminación, de ser posible sin ruido y con una silla cómoda.

Respeta los tiempos de comida
Es muy importante no descuidar la alimentación.
Organiza tu tiempo
Hacer un calendario semanal para organizar su tiempo entre las clases virtuales, la entrega de asignaciones y la preparación de los exámenes. No olvides tiempo para distraerte y hacer ejercicio.
Alista los materiales
Evita tener que buscar materiales que ocupes en medio de una clase o mientras estudias. Aprovecha al máximo tu tiempo.
Música relajante
Antes o durante el estudio puedes utilizar música relajante que te ayude a la concentración.
Aromaterapia
Puede llegar a ser un efectivo método para mantenerse relajado.
Toma awa UwU
Es importante mantenerse hidratado.
